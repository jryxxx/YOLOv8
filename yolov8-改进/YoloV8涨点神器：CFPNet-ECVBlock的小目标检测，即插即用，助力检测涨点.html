1.Centralized Feature Pyramid for Object Detection

论文地址： https://arxiv.org/abs/2210.02093

CFPNet即插即用，助力检测涨点，YOLOX/YOLOv5/YOLOV7均有效

摘要
视觉特征金字塔在广泛的应用中显示出其有效性和效率的优越性。

然而，现有的方法过分地集中于层间特征交互，而忽略了层内特征规则，这是经验证明是有益的。尽管一些方法试图在注意力机制或视觉变换器的帮助下学习紧凑的层内特征表示，但它们忽略了对密集预测任务非常重要的被忽略的角点区域。为了解决这一问题，本文提出了一种基于全局显式集中式特征规则的集中式特征金字塔（CFP）对象检测方法。具体而言，我们首先提出了一种空间显式视觉中心方案，其中使用轻量级MLP来捕捉全局长距离依赖关系，并使用并行可学习视觉中心机制来捕捉输入图像的局部角区域。在此基础上，我们以自上而下的方式对常用特征金字塔提出了一种全局集中的规则，其中使用从最深层内特征获得的显式视觉中心信息来调整正面浅层特征。与现有的特征金字塔相比，CFP不仅能够捕获全局长距离依赖关系，而且能够有效地获得全面但有区别的特征表示。在具有挑战性的MS-COCO上的实验结果验证了我们提出的CFP能够在最先进的YOLOv5和YOLOX目标检测基线上实现一致的性能增益。该代码发布于：CFPNet。

1.2 Centralized Feature Pyramid (CFP)

如图2所示，CFP主要由以下部分组成：输入图像、用于提取视觉特征金字塔的CNN主干、提出的显式视觉中心（EVC）、提出的全局集中规则（GCR）以及用于目标检测的去解耦head网络（由分类损失、回归损失和分割损失组成）。在图2中，EVC和GCR在提取的特征金字塔上实现。

1.3 Explicit Visual Center (EVC)


    提出的EVC主要由两个并行连接的块组成，其中使用轻量级MLP来捕获顶级特征的全局长期依赖性（即全局信息）。

 

 2.yolov8加入ECVBlock

如何将ECVBlock应用到yolov8是本文的关键，重点是增强用于这些检测器的特征金字塔的表示。

1）将ECVBlock添加到backbone或者是head在不同数据集的性能会不一致，比如本文添加到backbone，在NEU-DET钢材表面缺陷和道路缺陷如任务中取得的涨点也是不一样的；

2）比如在backbone添加的位置不同对最终的性能也是完全不一样的，这点也佐证了深度学习具有玄学，体现了调参的必要性，在不断的调参中自然会取得一定经验值；

  32. class Encoding(nn.Module):
  33.     def __init__(self, in_channels, num_codes):
  34.         super(Encoding, self).__init__()
  35.         # init codewords and smoothing factor
  36.         self.in_channels, self.num_codes = in_channels, num_codes
  37.         num_codes = 64
  38.         std = 1. / ((num_codes * in_channels)**0.5)
  39.         # [num_codes, channels]
  40.         self.codewords = nn.Parameter(
  41.             torch.empty(num_codes, in_channels, dtype=torch.float).uniform_(-std, std), requires_grad=True)
  42.         # [num_codes]
  43.         self.scale = nn.Parameter(torch.empty(num_codes, dtype=torch.float).uniform_(-1, 0), requires_grad=True)
  44.  
  45.     @staticmethod
  46.     def scaled_l2(x, codewords, scale):
  47.         num_codes, in_channels = codewords.size()
  48.         b = x.size(0)
  49.         expanded_x = x.unsqueeze(2).expand((b, x.size(1), num_codes, in_channels))
  50.  
  51.      
  52.         reshaped_codewords = codewords.view((1, 1, num_codes, in_channels))
  53.  
  54.        
  55.         reshaped_scale = scale.view((1, 1, num_codes))  # N, num_codes
  56.  
  57.      
  58.         scaled_l2_norm = reshaped_scale * (expanded_x - reshaped_codewords).pow(2).sum(dim=3)
  59.         return scaled_l2_norm
  60.  
  61.     @staticmethod
  62.     def aggregate(assignment_weights, x, codewords):
  63.         num_codes, in_channels = codewords.size()
  64.  
  65.       
  66.         reshaped_codewords = codewords.view((1, 1, num_codes, in_channels))
  67.         b = x.size(0)
  68.  
  69.        
  70.         expanded_x = x.unsqueeze(2).expand((b, x.size(1), num_codes, in_channels))
  71.  
  72.       
  73.         assignment_weights = assignment_weights.unsqueeze(3)  # b, N, num_codes,
  74.  
  75.        
  76.         encoded_feat = (assignment_weights * (expanded_x - reshaped_codewords)).sum(1)
  77.         return encoded_feat
  78.  
  79.     def forward(self, x):
  80.         assert x.dim() == 4 and x.size(1) == self.in_channels
  81.         b, in_channels, w, h = x.size()
  82.  
  83.         # [batch_size, height x width, channels]
  84.         x = x.view(b, self.in_channels, -1).transpose(1, 2).contiguous()
  85.  
  86.         # assignment_weights: [batch_size, channels, num_codes]
  87.         assignment_weights = F.softmax(self.scaled_l2(x, self.codewords, self.scale), dim=2)
  88.  
  89.         # aggregate
  90.         encoded_feat = self.aggregate(assignment_weights, x, self.codewords)
  91.         return encoded_feat

  95. class ConvBlock(nn.Module):
  96.     def __init__(self, in_channels, out_channels, stride=1, res_conv=False, act_layer=nn.ReLU, groups=1,
  97.                  norm_layer=partial(nn.BatchNorm2d, eps=1e-6), drop_block=None, drop_path=None):
  98.         super(ConvBlock, self).__init__()
  99.         self.in_channels = in_channels
  100.         expansion = 4
  101.         c = out_channels // expansion
  102.  
  103.         self.conv1 = nn.Conv2d(in_channels, c, kernel_size=1, stride=1, padding=0, bias=False)  # [64, 256, 1, 1]
  104.         self.bn1 = norm_layer(c)
  105.         self.act1 = act_layer(inplace=True)
  106.  
  107.         self.conv2 = nn.Conv2d(c, c, kernel_size=3, stride=stride, groups=groups, padding=1, bias=False)
  108.         self.bn2 = norm_layer(c)
  109.         self.act2 = act_layer(inplace=True)
  110.  
  111.         self.conv3 = nn.Conv2d(c, out_channels, kernel_size=1, stride=1, padding=0, bias=False)
  112.         self.bn3 = norm_layer(out_channels)
  113.         self.act3 = act_layer(inplace=True)
  114.  
  115.         if res_conv:
  116.             self.residual_conv = nn.Conv2d(in_channels, out_channels, kernel_size=1, stride=1, padding=0, bias=False)
  117.             self.residual_bn = norm_layer(out_channels)
  118.  
  119.         self.res_conv = res_conv
  120.         self.drop_block = drop_block
  121.         self.drop_path = drop_path
  122.  
  123.     def zero_init_last_bn(self):
  124.         nn.init.zeros_(self.bn3.weight)
  125.  
  126.     def forward(self, x, return_x_2=True):
  127.         residual = x
  128.  
  129.         x = self.conv1(x)
  130.         x = self.bn1(x)
  131.         if self.drop_block is not None:
  132.             x = self.drop_block(x)
  133.         x = self.act1(x)
  134.  
  135.         x = self.conv2(x) #if x_t_r is None else self.conv2(x + x_t_r)
  136.         x = self.bn2(x)
  137.         if self.drop_block is not None:
  138.             x = self.drop_block(x)
  139.         x2 = self.act2(x)
  140.  
  141.         x = self.conv3(x2)
  142.         x = self.bn3(x)
  143.         if self.drop_block is not None:
  144.             x = self.drop_block(x)
  145.  
  146.         if self.drop_path is not None:
  147.             x = self.drop_path(x)
  148.  
  149.         if self.res_conv:
  150.             residual = self.residual_conv(residual)
  151.             residual = self.residual_bn(residual)
  152.  
  153.         x += residual
  154.         x = self.act3(x)
  155.  
  156.         if return_x_2:
  157.             return x, x2
  158.         else:
  159.             return x

  172. class Mlp(nn.Module):
  173.     """
  174.     Implementation of MLP with 1*1 convolutions. Input: tensor with shape [B, C, H, W]
  175.     """
  176.     def __init__(self, in_features, hidden_features=None,
  177.                  out_features=None, act_layer=nn.GELU, drop=0.):
  178.         super().__init__()
  179.         out_features = out_features or in_features
  180.         hidden_features = hidden_features or in_features
  181.         self.fc1 = nn.Conv2d(in_features, hidden_features, 1)
  182.         self.act = act_layer()
  183.         self.fc2 = nn.Conv2d(hidden_features, out_features, 1)
  184.         self.drop = nn.Dropout(drop)
  185.         self.apply(self._init_weights)
  186.  
  187.     def _init_weights(self, m):
  188.         if isinstance(m, nn.Conv2d):
  189.             trunc_normal_(m.weight, std=.02)
  190.             if m.bias is not None:
  191.                 nn.init.constant_(m.bias, 0)
  192.  
  193.     def forward(self, x):
  194.         x = self.fc1(x)
  195.         x = self.act(x)
  196.         x = self.drop(x)
  197.         x = self.fc2(x)
  198.         x = self.drop(x)
  199.         return x

  202. class LayerNormChannel(nn.Module):
  203.     """
  204.     LayerNorm only for Channel Dimension.
  205.     Input: tensor in shape [B, C, H, W]
  206.     """
  207.     def __init__(self, num_channels, eps=1e-05):
  208.         super().__init__()
  209.         self.weight = nn.Parameter(torch.ones(num_channels))
  210.         self.bias = nn.Parameter(torch.zeros(num_channels))
  211.         self.eps = eps
  212.  
  213.     def forward(self, x):
  214.         u = x.mean(1, keepdim=True)
  215.         s = (x - u).pow(2).mean(1, keepdim=True)
  216.         x = (x - u) / torch.sqrt(s + self.eps)
  217.         x = self.weight.unsqueeze(-1).unsqueeze(-1) * x \
  218.             + self.bias.unsqueeze(-1).unsqueeze(-1)
  219.         return x

  249. class BaseConv(nn.Module):
  250.     """A Conv2d -> Batchnorm -> silu/leaky relu block"""  # CBL
  251.  
  252.     def __init__(
  253.             self, in_channels, out_channels, ksize, stride, groups=1, bias=False, act="silu"
  254.     ):
  255.         super().__init__()
  256.         # same padding
  257.         pad = (ksize - 1) // 2
  258.         self.conv = nn.Conv2d(
  259.             in_channels,
  260.             out_channels,
  261.             kernel_size=ksize,
  262.             stride=stride,
  263.             padding=pad,
  264.             groups=groups,
  265.             bias=bias,
  266.         )
  267.         self.bn = nn.BatchNorm2d(out_channels)
  268.         self.act = get_activation(act, inplace=True)
  269.  
  270.     def forward(self, x):
  271.         return self.act(self.bn(self.conv(x)))
  272.  
  273.     def fuseforward(self, x):
  274.         return self.act(self.conv(x))

  299. class LVCBlock(nn.Module):
  300.     def __init__(self, c1, c2, num_codes, channel_ratio=0.25, base_channel=64):
  301.         super(LVCBlock, self).__init__()
  302.         self.c2 = c2
  303.         self.num_codes = num_codes
  304.         num_codes = 64
  305.  
  306.         self.conv_1 = ConvBlock(c1, c1, res_conv=True, stride=1)
  307.  
  308.         self.LVC = nn.Sequential(
  309.             nn.Conv2d(c1, c1, 1, bias=False),
  310.             nn.BatchNorm2d(c1),
  311.             nn.ReLU(inplace=True),
  312.             Encoding(c1, num_codes=num_codes),
  313.             nn.BatchNorm1d(num_codes),
  314.             nn.ReLU(inplace=True),
  315.             Mean(dim=1))
  316.         self.fc = nn.Sequential(nn.Linear(c1, c1), nn.Sigmoid())
  317.  
  318.     def forward(self, x):
  319.         x = self.conv_1(x, return_x_2=False)
  320.         en = self.LVC(x)
  321.         gam = self.fc(en)
  322.         b, in_channels, _, _ = x.size()
  323.         y = gam.view(b, in_channels, 1, 1)
  324.         x = F.relu_(x + x * y)
  325.         return x

  329. class LightMLPBlock(nn.Module):
  330.     def __init__(self, c1, c2, ksize=1, stride=1, act="silu",
  331.                  mlp_ratio=4., drop=0., act_layer=nn.GELU,
  332.                  use_layer_scale=True, layer_scale_init_value=1e-5, drop_path=0.,
  333.                  norm_layer=GroupNorm):  # act_layer=nn.GELU,
  334.         super().__init__()
  335.         self.dw = DWConv(c1, c2, ksize=1, stride=1, act="silu")
  336.         self.linear = nn.Linear(c2, c2)  # learnable position embedding
  337.         self.c2 = c2
  338.  
  339.         self.norm1 = norm_layer(c1)
  340.         self.norm2 = norm_layer(c1)
  341.  
  342.         mlp_hidden_dim = int(c1 * mlp_ratio)
  343.         self.mlp = Mlp(in_features=c1, hidden_features=mlp_hidden_dim, act_layer=nn.GELU,
  344.                        drop=drop)
  345.  
  346.         self.drop_path = DropPath(drop_path) if drop_path > 0. \
  347.             else nn.Identity()
  348.  
  349.         self.use_layer_scale = use_layer_scale
  350.         if use_layer_scale:
  351.             self.layer_scale_1 = nn.Parameter(
  352.                 layer_scale_init_value * torch.ones((c2)), requires_grad=True)
  353.             self.layer_scale_2 = nn.Parameter(
  354.                 layer_scale_init_value * torch.ones((c2)), requires_grad=True)
  355.  
  356.     def forward(self, x):
  357.         if self.use_layer_scale:
  358.             x = x + self.drop_path(self.layer_scale_1.unsqueeze(-1).unsqueeze(-1) * self.dw(self.norm1(x)))
  359.             x = x + self.drop_path(self.layer_scale_2.unsqueeze(-1).unsqueeze(-1) * self.mlp(self.norm2(x)))
  360.         else:
  361.             x = x + self.drop_path(self.dw(self.norm1(x)))
  362.             x = x + self.drop_path(self.mlp(self.norm2(x)))
  363.         return x

  366. # EVCBlock
  367. class EVCBlock(nn.Module):
  368.     def __init__(self, c1, c2, channel_ratio=4, base_channel=16):
  369.         super().__init__()
  370.         expansion = 2
  371.         ch = c2 * expansion
  372.         # Stem stage: get the feature maps by conv block (copied form resnet.py) 进入conformer框架之前的处理
  373.         self.conv1 = nn.Conv2d(c1, c1, kernel_size=7, stride=1, padding=3, bias=False)  # 1 / 2 [112, 112]
  374.         self.bn1 = nn.BatchNorm2d(c1)
  375.         self.act1 = nn.ReLU(inplace=True)
  376.         self.maxpool = nn.MaxPool2d(kernel_size=3, stride=1, padding=1)  # 1 / 4 [56, 56]
  377.  
  378.         # LVC
  379.         self.lvc = LVCBlock(c1, c2, num_codes=64)  # c1值暂时未定
  380.         # LightMLPBlock
  381.         self.l_MLP = LightMLPBlock(c1, c2, ksize=1, stride=1, act="silu", act_layer=nn.GELU, mlp_ratio=4., drop=0.,
  382.                                    use_layer_scale=True, layer_scale_init_value=1e-5, drop_path=0.,
  383.                                    norm_layer=GroupNorm)
  384.         self.cnv1 = nn.Conv2d(ch, c2, kernel_size=1, stride=1, padding=0)
  385.  
  386.     def forward(self, x):
  387.         x1 = self.maxpool(self.act1(self.bn1(self.conv1(x))))
  388.         # LVCBlock
  389.         x_lvc = self.lvc(x1)
  390.         # LightMLPBlock
  391.         x_lmlp = self.l_MLP(x1)
  392.         # concat
  393.         x = torch.cat((x_lvc, x_lmlp), dim=1)
  394.         x = self.cnv1(x)
  395.         return x
