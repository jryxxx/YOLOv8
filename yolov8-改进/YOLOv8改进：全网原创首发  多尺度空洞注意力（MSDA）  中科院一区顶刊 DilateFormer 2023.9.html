1.DilateFormer介绍

论文： 2302.01791.pdf (arxiv.org)

收敛： 

本文提出了一种新颖的多尺度空洞 Transformer，简称DilateFormer，以用于视觉识别任务。原有的 ViT 模型在计算复杂性和感受野大小之间的权衡上存在矛盾。众所周知，ViT 模型使用全局注意力机制，能够在任意图像块之间建立长远距离上下文依赖关系，但是全局感受野带来的是平方级别的计算代价。同时，有些研究表明，在浅层特征上，直接进行全局依赖性建模可能存在冗余，因此是没必要的。

为了克服这些问题，作者提出了一种新的注意力机制——多尺度空洞注意力（MSDA）。MSDA 能够模拟小范围内的局部和稀疏的图像块交互，这些发现源自于对 ViTs 在浅层次上全局注意力中图像块交互的分析。作者发现在浅层次上，注意力矩阵具有局部性和稀疏性两个关键属性，这表明在浅层次的语义建模中，远离查询块的块大部分无关，因此全局注意力模块中存在大量的冗余。

DilateFormer 是一个以金字塔结构为基础的深度学习模型，它主要设计用来处理基础的视觉任务。DilateFormer 的关键设计概念是利用多尺度空洞注意力（Multi-Scale Dilated Attention, MSDA）来有效捕捉多尺度的语义信息，并减少自注意力机制的冗余。

如下图所示，MSDA 模块同样采用多头的设计，将特征图的通道分为 n 个不同的头部，并在不同的头部使用不同的空洞率执行滑动窗口膨胀注意力（SWDA）。这样可以在被关注的感受野内的各个尺度上聚合语义信息，并有效地减少自注意力机制的冗余，无需复杂的操作和额外的计算成本。

总体来说，DilateFormer 通过这种混合使用多尺度空洞注意力和多头自注意力的方式，成功地处理了长距离依赖问题，同时保持了计算效率，并能够适应不同尺度和分辨率的输入。

  26. class DilateAttention(nn.Module):
  27.     "Implementation of Dilate-attention"
  28.     def __init__(self, head_dim, qk_scale=None, attn_drop=0, kernel_size=3, dilation=1):
  29.         super().__init__()
  30.         self.head_dim = head_dim
  31.         self.scale = qk_scale or head_dim ** -0.5
  32.         self.kernel_size=kernel_size
  33.         self.unfold = nn.Unfold(kernel_size, dilation, dilation*(kernel_size-1)//2, 1)
  34.         self.attn_drop = nn.Dropout(attn_drop)
  35.  
  36.     def forward(self,q,k,v):
  37.         #B, C//3, H, W
  38.         B,d,H,W = q.shape
  39.  
  40.         q = q.reshape([B, d//self.head_dim, self.head_dim, 1 ,H*W]).permute(0, 1, 4, 3, 2)  # B,h,N,1,d
  41.         k = self.unfold(k).reshape([B, d//self.head_dim, self.head_dim, self.kernel_size*self.kernel_size, H*W]).permute(0, 1, 4, 2, 3)  #B,h,N,d,k*k
  42.         attn = (q @ k) * self.scale  # B,h,N,1,k*k
  43.         attn = attn.softmax(dim=-1)
  44.         attn = self.attn_drop(attn)
  45.         v = self.unfold(v).reshape([B, d//self.head_dim, self.head_dim, self.kernel_size*self.kernel_size, H*W]).permute(0, 1, 4, 3, 2)  # B,h,N,k*k,d
  46.         x = (attn @ v).transpose(1, 2).reshape(B, H, W, d)
  47.         return x

  50. class MultiDilatelocalAttention(nn.Module):
  51.     "Implementation of Dilate-attention"
  52.  
  53.     def __init__(self, dim, num_heads=4, qkv_bias=False, qk_scale=None,
  54.                  attn_drop=0.,proj_drop=0., kernel_size=3, dilation=[1, 2]):
  55.         super().__init__()
  56.         self.dim = dim
  57.         self.num_heads = num_heads
  58.         head_dim = dim // num_heads
  59.         self.dilation = dilation
  60.         self.kernel_size = kernel_size
  61.         self.scale = qk_scale or head_dim ** -0.5
  62.         self.num_dilation = len(dilation)
  63.  
  64.         assert num_heads % self.num_dilation == 0, f"num_heads{num_heads} must be the times of num_dilation{self.num_dilation}!!"
  65.         self.qkv = nn.Conv2d(dim, dim * 3, 1, bias=qkv_bias)
  66.         self.dilate_attention = nn.ModuleList(
  67.             [DilateAttention(head_dim, qk_scale, attn_drop, kernel_size, dilation[i])
  68.              for i in range(self.num_dilation)])
  69.         self.proj = nn.Linear(dim, dim)
  70.         self.proj_drop = nn.Dropout(proj_drop)
  71.  
  72.     def forward(self, x):
  73.  
  74.         x = x.permute(0, 3, 1, 2)  # B, C, H, W
  75.         B, C, H, W = x.shape
  76.  
  77.         qkv = self.qkv(x).reshape(B, 3, self.num_dilation, C //self.num_dilation, H, W).permute(2, 1, 0, 3, 4, 5)
  78.         #num_dilation,3,B,C//num_dilation,H,W
  79.         x = x.reshape(B, self.num_dilation, C//self.num_dilation, H, W).permute(1, 0, 3, 4, 2 )
  80.         # num_dilation, B, H, W, C//num_dilation
  81.         for i in range(self.num_dilation):
  82.             x[i] = self.dilate_attention[i](qkv[i][0], qkv[i][1], qkv[i][2])# B, H, W,C//num_dilation
  83.         x = x.permute(1, 2, 3, 0, 4).reshape(B, H, W, C)
  84.         x = self.proj(x)
  85.         x = self.proj_drop(x)
  86.         return x

  89. class DilateBlock(nn.Module):
  90.     "Implementation of Dilate-attention block"
  91.     def __init__(self, dim, num_heads=4, mlp_ratio=4., qkv_bias=False,qk_scale=None, drop=0., attn_drop=0.,
  92.                  drop_path=0.,act_layer=nn.GELU, norm_layer=nn.LayerNorm, kernel_size=3, dilation=[1, 2],
  93.                  cpe_per_block=False):
  94.         super().__init__()
  95.         self.dim = dim
  96.         self.num_heads = num_heads
  97.         self.mlp_ratio = mlp_ratio
  98.         self.kernel_size = kernel_size
  99.         self.dilation = dilation
  100.         self.cpe_per_block = cpe_per_block
  101.         if self.cpe_per_block:
  102.             self.pos_embed = nn.Conv2d(dim, dim, 3, padding=1, groups=dim)
  103.         self.norm1 = norm_layer(dim)
  104.         self.attn = MultiDilatelocalAttention(dim, num_heads=num_heads, qkv_bias=qkv_bias, qk_scale=qk_scale,
  105.                                                 attn_drop=attn_drop, kernel_size=kernel_size, dilation=dilation)
  106.  
  107.         self.drop_path = DropPath(
  108.             drop_path) if drop_path > 0. else nn.Identity()
  109.

  113.  
  114.         x = x.permute(0, 3, 2, 1)
  115.         x = x + self.drop_path(self.attn(self.norm1(x)))
  116.         x = x.permute(0, 3, 2, 1)
  117.         #B, C, H, W

  121. def autopad(k, p=None, d=1):  # kernel, padding, dilation
  122.     # Pad to 'same' shape outputs
  123.     if d > 1:
  124.         k = d * (k - 1) + 1 if isinstance(k, int) else [d * (x - 1) + 1 for x in k]  # actual kernel-size
  125.     if p is None:
  126.         p = k // 2 if isinstance(k, int) else [x // 2 for x in k]  # auto-pad
  127.     return p

  130. class Conv(nn.Module):
  131.     # Standard convolution with args(ch_in, ch_out, kernel, stride, padding, groups, dilation, activation)
  132.     default_act = nn.SiLU()  # default activation
  133.  
  134.     def __init__(self, c1, c2, k=1, s=1, p=None, g=1, d=1, act=True):
  135.         super().__init__()
  136.         self.conv = nn.Conv2d(c1, c2, k, s, autopad(k, p, d), groups=g, dilation=d, bias=False)
  137.         self.bn = nn.BatchNorm2d(c2)
  138.         self.act = self.default_act if act is True else act if isinstance(act, nn.Module) else nn.Identity()
  139.  
  140.     def forward(self, x):
  141.         return self.act(self.bn(self.conv(x)))
  142.  
  143.     def forward_fuse(self, x):
  144.         return self.act(self.conv(x))

  146. class C2f(nn.Module):
  147.     # CSP Bottleneck with 2 convolutions
  148.     def __init__(self, c1, c2, n=1, shortcut=False, g=1, e=0.5):  # ch_in, ch_out, number, shortcut, groups, expansion
  149.         super().__init__()
  150.         self.c = int(c2 * e)  # hidden channels
  151.         self.cv1 = Conv(c1, 2 * self.c, 1, 1)
  152.         self.cv2 = Conv((2 + n) * self.c, c2, 1)  # optional act=FReLU(c2)
  153.         self.m = nn.ModuleList(Bottleneck(self.c, self.c, shortcut, g, k=((3, 3), (3, 3)), e=1.0) for _ in range(n))
  154.  
  155.     def forward(self, x):
  156.         y = list(self.cv1(x).chunk(2, 1))
  157.         y.extend(m(y[-1]) for m in self.m)
  158.         return self.cv2(torch.cat(y, 1))
  159.  
  160.     def forward_split(self, x):
  161.         y = list(self.cv1(x).split((self.c, self.c), 1))
  162.         y.extend(m(y[-1]) for m in self.m)
  163.         return self.cv2(torch.cat(y, 1))
  164.  
  165. class Bottleneck(nn.Module):
  166.     # Standard bottleneck
  167.     def __init__(self, c1, c2, shortcut=True, g=1, k=(3, 3), e=0.5):  # ch_in, ch_out, shortcut, groups, kernels, expand
  168.         super().__init__()
  169.         c_ = int(c2 * e)  # hidden channels
  170.         self.cv1 = Conv(c1, c_, k[0], 1)
  171.         self.cv2 = Conv(c_, c2, k[1], 1, g=g)
  172.         self.add = shortcut and c1 == c2
  173.  
  174.     def forward(self, x):
  175.         return x + self.cv2(self.cv1(x)) if self.add else self.cv2(self.cv1(x))
  176.

  178.     # CSP Bottleneck with 2 convolutions
  179.     def __init__(self, c1, c2, n=1, shortcut=False, g=1, e=0.5):  # ch_in, ch_out, number, shortcut, groups, expansion
  180.         super().__init__()
  181.         self.c = int(c2 * e)  # hidden channels
  182.         self.cv1 = Conv(c1, 2 * self.c, 1, 1)
  183.         self.cv2 = Conv((2 + n) * self.c, c2, 1)  # optional act=FReLU(c2)
  184.         self.m = nn.ModuleList(DilateBlock(self.c) for _ in range(n))
  185.

  188.         y.extend(m(y[-1]) for m in self.m)
  189.         return self.cv2(torch.cat(y, 1))

  193.         y.extend(m(y[-1]) for m in self.m)
  194.         return self.cv2(torch.cat(y, 1))
