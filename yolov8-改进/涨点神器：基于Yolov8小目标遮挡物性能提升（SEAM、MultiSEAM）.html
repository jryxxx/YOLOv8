1.遮挡物检测简介 

不同的目标检测应用场景有不同的检测难点，小目标、多尺度以及背景复杂等问题，被遮挡的物体仍然是最先进的物体检测器面临的挑战。本文尝试解决待测目标相互遮挡带来的检测困难，对于人脸遮挡提出了一个名为 SEAM 的注意力模块并引入了排斥损失来解决它，引入了分离和增强注意力模块来增强Neck层输出部分后被遮挡人脸的响应能力。

1.1 Separated and Enhancement Attention Module (SEAM)

即不同人脸之间的遮挡，以及其他物体对人脸的遮挡。前者使得检测精度对 NMS 阈值非常敏感，从而导致漏检。作者使用排斥损失进行人脸检测，它惩罚预测框转移到其他真实目标，并要求每个预测框远离具有不同指定目标的其他预测框，以使检测结果对 NMS 不太敏感。后者导致特征消失导致定位不准确，设计了注意力模块 SEAM 来增强人脸特征的学习。

  SEAM的第一部分是带有残差连接的深度可分离卷积。深度可分离卷积是逐个深度操作的，即逐个通道分离卷积。深度可分离卷积虽然可以学习不同通道的重要性并减少参数量，但它忽略了通道之间的信息关系。

     为了弥补这种损失，不同深度卷积的输出随后通过逐点卷积进行组合。然后用一个两层的全连接网络来融合每个通道的信息，这样网络就可以加强所有通道之间的连接。

 1.2 MultiSEAM

    解决多尺度问题的主要方法是构建金字塔来融合人脸的多尺度特征。例如，在 YOLOv5 中，FPN 融合了 P3、P4 和 P5 层的特征。但是对于小尺度的目标，经过多层卷积后信息很容易丢失，保留的像素信息很少，即使在较浅的P3层也是如此。因此，提高特征图的分辨率无疑有利于小目标的检测。

1.3遮挡感知排斥损失

 RepGT Loss 的作用是使当前边界框尽可能远离周围的ground truth box。 这里的周围ground truth box是指除了bounding box本身要返回的对象之外，与人脸IoU最大的人脸标签。

  2. class TridentBlock(nn.Module):
  3.     def __init__(self, c1, c2, stride=1, c=False, e=0.5, padding=[1, 2, 3], dilate=[1, 2, 3], bias=False):
  4.         super(TridentBlock, self).__init__()
  5.         self.stride = stride
  6.         self.c = c
  7.         c_ = int(c2 * e)
  8.         self.padding = padding
  9.         self.dilate = dilate
  10.         self.share_weightconv1 = nn.Parameter(torch.Tensor(c_, c1, 1, 1))
  11.         self.share_weightconv2 = nn.Parameter(torch.Tensor(c2, c_, 3, 3))
  12.  
  13.         self.bn1 = nn.BatchNorm2d(c_)
  14.         self.bn2 = nn.BatchNorm2d(c2)
  15.  
  16.         self.act = nn.SiLU()
  17.  
  18.         nn.init.kaiming_uniform_(self.share_weightconv1, nonlinearity="relu")
  19.         nn.init.kaiming_uniform_(self.share_weightconv2, nonlinearity="relu")
  20.  
  21.         if bias:
  22.             self.bias = nn.Parameter(torch.Tensor(c2))
  23.         else:
  24.             self.bias = None
  25.  
  26.         if self.bias is not None:
  27.             nn.init.constant_(self.bias, 0)
  28.  
  29.     def forward_for_small(self, x):
  30.         residual = x
  31.         out = nn.functional.conv2d(x, self.share_weightconv1, bias=self.bias)
  32.         out = self.bn1(out)
  33.         out = self.act(out)
  34.  
  35.         out = nn.functional.conv2d(out, self.share_weightconv2, bias=self.bias, stride=self.stride, padding=self.padding[0],
  36.                                    dilation=self.dilate[0])
  37.         out = self.bn2(out)
  38.         out += residual
  39.         out = self.act(out)
  40.  
  41.         return out
  42.  
  43.     def forward_for_middle(self, x):
  44.         residual = x
  45.         out = nn.functional.conv2d(x, self.share_weightconv1, bias=self.bias)
  46.         out = self.bn1(out)
  47.         out = self.act(out)
  48.  
  49.         out = nn.functional.conv2d(out, self.share_weightconv2, bias=self.bias, stride=self.stride, padding=self.padding[1],
  50.                                    dilation=self.dilate[1])
  51.         out = self.bn2(out)
  52.         out += residual
  53.         out = self.act(out)
  54.  
  55.         return out
  56.  
  57.     def forward_for_big(self, x):
  58.         residual = x
  59.         out = nn.functional.conv2d(x, self.share_weightconv1, bias=self.bias)
  60.         out = self.bn1(out)
  61.         out = self.act(out)
  62.  
  63.         out = nn.functional.conv2d(out, self.share_weightconv2, bias=self.bias, stride=self.stride, padding=self.padding[2],
  64.                                    dilation=self.dilate[2])
  65.         out = self.bn2(out)
  66.         out += residual
  67.         out = self.act(out)
  68.  
  69.         return out
  70.  
  71.     def forward(self, x):
  72.         xm = x
  73.         base_feat = []
  74.         if self.c is not False:
  75.             x1 = self.forward_for_small(x)
  76.             x2 = self.forward_for_middle(x)
  77.             x3 = self.forward_for_big(x)
  78.         else:
  79.             x1 = self.forward_for_small(xm[0])
  80.             x2 = self.forward_for_middle(xm[1])
  81.             x3 = self.forward_for_big(xm[2])
  82.  
  83.         base_feat.append(x1)
  84.         base_feat.append(x2)
  85.         base_feat.append(x3)
  86.  
  87.         return base_feat

  89. class RFEM(nn.Module):
  90.     def __init__(self, c1, c2, n=1, e=0.5, stride=1):
  91.         super(RFEM, self).__init__()
  92.         c = True
  93.         layers = []
  94.         layers.append(TridentBlock(c1, c2, stride=stride, c=c, e=e))
  95.         c1 = c2
  96.         for i in range(1, n):
  97.             layers.append(TridentBlock(c1, c2))
  98.         self.layer = nn.Sequential(*layers)
  99.         # self.cv = Conv(c2, c2)
  100.         self.bn = nn.BatchNorm2d(c2)
  101.         self.act = nn.SiLU()
  102.  
  103.     def forward(self, x):
  104.         out = self.layer(x)
  105.         out = out[0] + out[1] + out[2] + x
  106.         out = self.act(self.bn(out))
  107.         return out
  108.  
  109. class Residual(nn.Module):
  110.     def __init__(self, fn):
  111.         super(Residual, self).__init__()
  112.         self.fn = fn
  113.  
  114.     def forward(self, x):
  115.         return self.fn(x) + x
  116. class ConvMixer(nn.Module):
  117.     def __init__(self, c1, c2, depth, kernel_size=3, patch_size=4, reduction=16):
  118.         super(ConvMixer, self).__init__()
  119.         if c1 != c2:
  120.             c2 = c1
  121.         self.DConvN = nn.Sequential(
  122.             nn.Conv2d(c1, c2, kernel_size=patch_size, stride=patch_size),
  123.             nn.GELU(),
  124.             nn.BatchNorm2d(c2),
  125.             *[nn.Sequential(
  126.                 Residual(nn.Sequential(
  127.                     nn.Conv2d(c2, c2, kernel_size, groups=c2, padding=1),
  128.                     nn.GELU(),
  129.                     nn.BatchNorm2d(c2)
  130.                 )),
  131.                 nn.Conv2d(c2, c1, kernel_size=1),
  132.                 nn.GELU(),
  133.                 nn.BatchNorm2d(c2)
  134.             ) for i in range(depth)]
  135.         )
  136.         self.avg_pool = torch.nn.AdaptiveAvgPool2d(1)
  137.         self.fc = nn.Sequential(
  138.             nn.Linear(c2, c2 // reduction, bias=False),
  139.             nn.ReLU(inplace=True),
  140.             nn.Linear(c2 // reduction, c2, bias=False),
  141.             nn.Sigmoid()
  142.         )
  143.  
  144.     def forward(self, x):
  145.         b, c, _, _ = x.size()
  146.         y = self.DConvN(x)
  147.         y = self.avg_pool(y).view(b, c)
  148.         y = self.fc(y).view(b, c, 1, 1)
  149.         y = torch.exp(y)
  150.         return x * y.expand_as(x)
  151.

  153. class SEAM(nn.Module):
  154.     def __init__(self, c1, c2, n, reduction=16):
  155.         super(SEAM, self).__init__()
  156.         if c1 != c2:
  157.             c2 = c1
  158.         self.DCovN = nn.Sequential(
  159.             # nn.Conv2d(c1, c2, kernel_size=3, stride=1, padding=1, groups=c1),
  160.             # nn.GELU(),
  161.             # nn.BatchNorm2d(c2),
  162.             *[nn.Sequential(
  163.                 Residual(nn.Sequential(
  164.                     nn.Conv2d(in_channels=c2, out_channels=c2, kernel_size=3, stride=1, padding=1, groups=c2),
  165.                     nn.GELU(),
  166.                     nn.BatchNorm2d(c2)
  167.                 )),
  168.                 nn.Conv2d(in_channels=c2, out_channels=c2, kernel_size=1, stride=1, padding=0, groups=1),
  169.                 nn.GELU(),
  170.                 nn.BatchNorm2d(c2)
  171.             ) for i in range(n)]
  172.         )
  173.         self.avg_pool = torch.nn.AdaptiveAvgPool2d(1)
  174.         self.fc = nn.Sequential(
  175.             nn.Linear(c2, c2 // reduction, bias=False),
  176.             nn.ReLU(inplace=True),
  177.             nn.Linear(c2 // reduction, c2, bias=False),
  178.             nn.Sigmoid()
  179.         )
  180.  
  181.         self._initialize_weights()
  182.         # self.initialize_layer(self.avg_pool)
  183.         self.initialize_layer(self.fc)
  184.  
  185.  
  186.     def forward(self, x):
  187.         b, c, _, _ = x.size()
  188.         y = self.DCovN(x)
  189.         y = self.avg_pool(y).view(b, c)
  190.         y = self.fc(y).view(b, c, 1, 1)
  191.         y = torch.exp(y)
  192.         return x * y.expand_as(x)
  193.  
  194.     def _initialize_weights(self):
  195.         for m in self.modules():
  196.             if isinstance(m, nn.Conv2d):
  197.                 nn.init.xavier_uniform_(m.weight, gain=1)
  198.             elif isinstance(m, nn.BatchNorm2d):
  199.                 nn.init.constant_(m.weight, 1)
  200.                 nn.init.constant_(m.bias, 0)
  201.  
  202.     def initialize_layer(self, layer):
  203.         if isinstance(layer, (nn.Conv2d, nn.Linear)):
  204.             torch.nn.init.normal_(layer.weight, mean=0., std=0.001)
  205.             if layer.bias is not None:
  206.                 torch.nn.init.constant_(layer.bias, 0)
  207.  
  208. def DcovN(c1, c2, depth, kernel_size=3, patch_size=3):
  209.     dcovn = nn.Sequential(
  210.         nn.Conv2d(c1, c2, kernel_size=patch_size, stride=patch_size),
  211.         nn.SiLU(),
  212.         nn.BatchNorm2d(c2),
  213.         *[nn.Sequential(
  214.             Residual(nn.Sequential(
  215.                 nn.Conv2d(in_channels=c2, out_channels=c2, kernel_size=kernel_size, stride=1, padding=1, groups=c2),
  216.                 nn.SiLU(),
  217.                 nn.BatchNorm2d(c2)
  218.             )),
  219.             nn.Conv2d(in_channels=c2, out_channels=c2, kernel_size=1, stride=1, padding=0, groups=1),
  220.             nn.SiLU(),
  221.             nn.BatchNorm2d(c2)
  222.         ) for i in range(depth)]
  223.     )
  224.     return dcovn

  226. class MultiSEAM(nn.Module):
  227.     def __init__(self, c1, c2, depth, kernel_size=3, patch_size=[3, 5, 7], reduction=16):
  228.         super(MultiSEAM, self).__init__()
  229.         if c1 != c2:
  230.             c2 = c1
  231.         self.DCovN0 = DcovN(c1, c2, depth, kernel_size=kernel_size, patch_size=patch_size[0])
  232.         self.DCovN1 = DcovN(c1, c2, depth, kernel_size=kernel_size, patch_size=patch_size[1])
  233.         self.DCovN2 = DcovN(c1, c2, depth, kernel_size=kernel_size, patch_size=patch_size[2])
  234.         self.avg_pool = torch.nn.AdaptiveAvgPool2d(1)
  235.         self.fc = nn.Sequential(
  236.             nn.Linear(c2, c2 // reduction, bias=False),
  237.             nn.ReLU(inplace=True),
  238.             nn.Linear(c2 // reduction, c2, bias=False),
  239.             nn.Sigmoid()
  240.         )
  241.  
  242.     def forward(self, x):
  243.         b, c, _, _ = x.size()
  244.         y0 = self.DCovN0(x)
  245.         y1 = self.DCovN1(x)
  246.         y2 = self.DCovN2(x)
  247.         y0 = self.avg_pool(y0).view(b, c)
  248.         y1 = self.avg_pool(y1).view(b, c)
  249.         y2 = self.avg_pool(y2).view(b, c)
  250.         y4 = self.avg_pool(x).view(b, c)
  251.         y = (y0 + y1 + y2 + y4) / 4
  252.         y = self.fc(y).view(b, c, 1, 1)
  253.         y = torch.exp(y)
  254.         return x * y.expand_as(x)

  257. class C3RFEM(nn.Module):
  258.     def __init__(self, c1, c2, n=1, shortcut=True, e=0.5):  # ch_in, ch_out, number, shortcut, groups, expansion
  259.         super().__init__()
  260.         c_ = int(c2 * e)  # hidden channels
  261.         self.cv1 = Conv(c1, c_, 1, 1)
  262.         self.cv2 = Conv(c1, c_, 1, 1)
  263.         self.cv3 = Conv(2 * c_, c2, 1)  # act=FReLU(c2)
  264.         # self.m = nn.Sequential(*(Bottleneck(c_, c_, shortcut, g, e=1.0) for _ in range(n)))
  265.         # self.rfem = RFEM(c_, c_, n)
  266.         self.m = nn.Sequential(*[RFEM(c_, c_, n=1, e=e) for _ in range(n)])
  267.         # self.m = nn.Sequential(*[CrossConv(c_, c_, 3, 1, g, 1.0, shortcut) for _ in range(n)])
  268.  
  269.     def forward(self, x):
  270.         return self.cv3(torch.cat((self.m(self.cv1(x)), self.cv2(x)), dim=1))
